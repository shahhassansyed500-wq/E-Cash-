EMoneyCashExchange is a secure financial platform designed to seamlessly convert digital e-money into real cash through verified agents. The system ensures fast transactions, user authentication, wallet verification, and transparent cash-out services — bringing digital finance closer to everyday convenience.
Features
• Secure login and user authentication • Customer wallet verification • Real-time cash-out requests • Role-based access (Admin, Agent, Customer) • Agent verification and transaction approvals • Transparent history tracking & receipts • Fast and reliable service process
Tech Stack
• Frontend: HTML, CSS, JavaScript (React/Flutter based on final tech) • Backend:Flutter, Firebase • Database:MongoDBB • Version Control: Git & GitHub
How It Works
Customer logs in & registers their e-wallet details
Customer requests a cash-out amount
Verified agent receives request and approves
Customer receives cash directly
System records transaction for transparency
Security Measures

• Encrypted user data • Secure transaction validation • Role-based access control • Agent identity verification
